# Quiz App 📝


## Features

- ✅ Multiple-choice questions
- ✅ Score tracking
- ✅ Timer tracking
- ✅ Leaderboard
- ✅ Randomized questions (over 4,146 questions)
- ✅ A lot of categories
- ✅ Choose between multiple difficulty levels
- ✅ User-friendly interface
